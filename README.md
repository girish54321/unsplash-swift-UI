# Unsplash-swift-UI

Welcome to the open source Unsplash app repository! This app is built using SwiftUI and Alamofire, and follows the Model-View-ViewModel (MVVM) design pattern. With this app, users can take advantage of the following features:

# Features

## Search images

Users can search for specific images by keyword or phrase using the app's built-in search function.
## View trending images
The app allows users to browse a collection of popular images from Unsplash.
## Search any image
Users can search for any image on Unsplash, giving them access to a vast collection of high quality images.

## Download images
The app enables users to download images from Unsplash to their device for offline viewing or to use in their own projects.

## Save to local storage 
Users can save images to their device's local storage for easy access in the future.
